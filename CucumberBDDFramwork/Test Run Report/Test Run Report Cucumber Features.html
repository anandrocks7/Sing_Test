Feature: TODO List Scenarios
Scenario: Add first item to empty List
  1. Given User is on the ToDo list Page
  2. When user add first item "Test1"
  3. Then only that item is displayed
  4. And the list summary shows "1 item left"
  5. And the lists selection is set to All
  6. And Clear completed button is not displayed
Scenario: Add Two item to empty List
  1. Given User is on the ToDo list Page
  2. When user add Two items
  3. Then only those item are displayed
  4. And the list summary shows "2 items left"
  5. And the lists selection is set to All
  6. And Clear completed button is not displayed
Scenario: Mark Item as completed and check summary
  1. Given Todo list with two or more items
  2. When the first item is marked as complete
  3. Then only the second item is listed as active
  4. And the list summary shows "1 item left"
  5. And Clear completed button is displayed
Scenario: Completed item is visible under completed filter
  1. Given Todo list with two or more items
  2. When the first item is marked as complete
  3. And user clicks on Completed filter
  4. Then only the completed item is displayed
Scenario: Completed item is not visible under Active filter
  1. Given Todo list with two or more items
  2. When the first item is marked as complete
  3. And user clicks on Active filter
  4. Then the completed item is not displayed
Scenario: Click on clear completed button and verify
  1. Given Todo list with two or more items
  2. When the first item is marked as complete
  3. And user clicks on Clear Completed button
  4. Then the completed item is not displayed
